# 替换词源文件（可维护版本）
#
# 格式：每行一条规则，"源词|目标词"
#   - 源词 = ASR 实际识别出来的文本
#   - 目标词 = 希望替换成的文本
#   - 用 | 分隔（与火山控制台 TXT 上传格式一致）
#
# 以 # 开头的行是注释，空行会被忽略，都不会进入上传文件。
# 行首行尾的空白会被自动去除。
# 同一个源词出现多次会告警（后者覆盖前者，以最后一条为准）。
#
# 生成上传文件：python3.10 replacements.py
# 仅校验不输出：python3.10 replacements.py --check
# 生成的文件为 replacements_upload.txt，到火山控制台手动上传。
# 上传后在控制台把词表绑定到对应的 ASR 应用/实例即可生效。

# ---- 通用纠错 ----
的的|的
了了|了
是是|是
